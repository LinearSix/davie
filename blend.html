The Brakumo band website makes use of a Postgres datbase to dynamically populate pages with current show, blog, and media information. Users can also add their information to a mailing list database table via a web form.

Data manipulation is accomplished via web forms by a logged-in admin.

Technologies:

  • Node.js
  • Express Server
  • Postgres
  • JavaScript
  • Embedded JavaScript (ejs)
  • Knex.js
  • NodeMailer
  • HTML
  • CSS
